Elevated Mode (2FA)
When enabled, authenticated users can unlock full Agent Zero access from Discord using !auth <key>. Default mode is restricted (chat only, no tools).
SECURITY NOTICE
Elevated mode grants Discord users full access to Agent Zero's capabilities, including code execution, file operations, and system tools. Before enabling, you must read and understand the security section in the plugin's README.md (located in the plugin directory or on the plugin's GitHub page).

Recommended configuration: A private server with only trusted members, a defined User Allowlist above, and short session timeouts. Only allow users you deeply trust.
